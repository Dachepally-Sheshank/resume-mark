# ResumeMark Product Specification

## Problem

Bookmarks always open the starting page.

Example:
Naruto bookmark opens Episode 1.

The user watches until Episode 82.

Next day they must manually find Episode 82 again.

## Solution

Automatically update the bookmark URL to the last visited page.

## Users

- Anime watchers
- Manga readers
- Online learners
- Documentation readers

## Success Criteria

One click resumes content from the last page visited.
